To run Sagasu, You need to install this Selenium. This is how you do it:

pip3 install selenium

You will also need Mozilla's GeckoDriver (from their GitHub) in order to run the Search command.

https://github.com/mozilla/geckodriver/releases

You will also need Firefox or any Firefox forks.

and finally, you will also need tkinter. here's how you install it:

pip3 install tk

Hope you read this README document and have fun building Sagasu!
